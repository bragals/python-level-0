## Em comum na lista

```
# Lista numéricas A e B
a = [1, 2, 3, 4, 5, 1]
b = [6, 7, 8, 9, 10, 1, 2, 11, 22]

# Verifica a quantidade de elementos que existem em comum entre as duas listas
comum = len(set(a).intersection(b))
print(comum)
```

### Output
`2`
